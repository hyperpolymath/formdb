== Event Sourcing

FormDB's journal-first design is naturally compatible with event sourcing:

[source,text]
----
FormDB Journal Entry = Event
───────────────────────────
{
  "sequence": 1001,
  "op_type": "DOC_INSERT",
  "collection": "evidence",
  "data": {...},
  "provenance": {
    "actor": "alice",
    "rationale": "Primary source"
  }
}
----
