== Development Setup

=== Prerequisites

* Gforth 0.7.3+
* Factor 0.99+
* Zig 0.11+
* Elixir 1.15+ (optional, for control plane)

=== Quick Start

[source,bash]
----
git clone https://github.com/hyperpolymath/formdb.git
cd formdb
just dev
----

=== Recommended Architecture

[source,text]
----
┌─────────────────────────────────────────┐
│  Load Balancer                          │
├─────────────────────────────────────────┤
│  Form.ControlPlane (Elixir/OTP)         │
│  ├── Session management                 │
│  └── Cluster coordination               │
├─────────────────────────────────────────┤
│  FormDB Core (Forth + Factor + Zig)     │
├─────────────────────────────────────────┤
│  Storage (Local SSD or Network Block)   │
└─────────────────────────────────────────┘
----
